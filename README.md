## About

Collaborative whiteboard for drawing and diagramming.

## Features

- Drawing tools
- Shapes
- Text
- Collaboration
- Export

## Installation

### From App Store
Open the **App Store** in zOS and search for "tldraw".

### Via npm
```bash
npm install @zos-apps/tldraw
```

## Usage

- Select tool from sidebar
- Draw on canvas
- Share to collaborate



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
